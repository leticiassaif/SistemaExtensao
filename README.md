# Sistema de Gerenciamento de Extensão — UFMA

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos, implementando em Java um sistema de gerenciamento de atividades de extensão universitária.

## Pré-requisitos

- Java JDK 11 ou superior
- Terminal (qualquer um)

Depois de compilar, para executar:

```bash
java -cp bin br.ufma.extensao.Main
```

## O que o sistema faz

O sistema simula o fluxo completo de extensão universitária — desde o cadastro de usuários e criação de oportunidades, passando por inscrições e execução, até a emissão de certificados e aproveitamento de horas externas. Grupos de extensão também podem ser criados e gerenciados.
